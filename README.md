## 📖 درباره پروژه

**Learning Academy** یک پلتفرم SaaS آموزش زبان آلمانی است که با ترکیب **هوش مصنوعی**، **گیمیفیکیشن** و **سیستم یادگیری تطبیقی**، تجربه‌ای جذاب و مؤثر برای زبان‌آموزان فراهم می‌کند.

### ✨ ویژگی‌های کلیدی

| ویژگی                      | توضیح                                                           |
| -------------------------- | --------------------------------------------------------------- |
| 🤖 **معلم هوش مصنوعی**     | مکالمه با AI (Gemini + OpenRouter) با قابلیت تصحیح گرامر و تلفظ |
| 📚 **درس‌های ساختاریافته** | ۱۲ درس کامل سطح A1 با محتوای غنی و تمرینات تعاملی               |
| 🎮 **گیمیفیکیشن**          | سیستم XP، Streak، Achievements و Leaderboard برای افزایش انگیزه |
| 📖 **دیکشنری هوشمند**      | جستجو، ذخیره لغات و فلش‌کارت با سیستم مرور فاصله‌دار            |
| 💬 **سناریوهای واقعی**     | مکالمه در موقعیت‌های روزمره با بازخورد هوش مصنوعی               |
| 🌍 **چندزبانه**            | پشتیبانی کامل از فارسی، انگلیسی و آلمانی                        |
| 🎨 **طراحی مدرن**          | Dark Mode، RTL/LTR، انیمیشن‌های روان و واکنش‌گرا                |
| 🔒 **امنیت**               | احراز هویت JWT با Refresh Token، Rate Limiting و Helmet         |
