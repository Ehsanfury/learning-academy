# Learning Academy Icons

This directory contains PWA icons for the Learning Academy application.

## Required Icon Sizes

| Size    | Purpose                |
| ------- | ---------------------- |
| 72x72   | Android launcher       |
| 96x96   | Android launcher       |
| 128x128 | Chrome extension       |
| 144x144 | Android launcher       |
| 152x152 | Apple touch icon       |
| 192x192 | Android launcher / PWA |
| 384x384 | PWA splash screen      |
| 512x512 | PWA splash screen      |

## Generating Icons

You can generate icons using:

1. **Favicon Generator**: https://realfavicongenerator.net/
2. **PWA Asset Generator**: https://www.pwabuilder.com/
3. **Image Editor**: Use any image editor to resize a 512x512 logo
